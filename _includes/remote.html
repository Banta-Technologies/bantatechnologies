Remote Services Setup

Go to https://rustdesk.com
Click the download button
Use the dropdown arrow next to the file name at the bottom of the browser
Or
Bring up the File Browser and click Downloads
Right click on the rustdesk zipped file and select extract
Navigate into the rustdesk folder and double click rustdesk.exe
Click Yes or OK to any promts for admin access
Follow the instructions to install Rust Desk
Start RustDesk by clicking the icon on your desktop
Click the menu button that looks like three dots

Enter rustdesk.bantatechnologies.com in the ID textbox
